# AutoRentar - Brand Guidelines

## Nombre Oficial
**AutoRentar** - "Alquiler Entre Personas"

## Paleta de Colores

### Color Principal (CTA)
| Nombre | Hex | Uso |
|--------|-----|-----|
| Verde Neon Radioactivo | `#39FF14` | Botones principales, CTAs, badges de exito |
| Verde Hover | `#2ECC11` | Hover states |
| Verde Pressed | `#25A50E` | Pressed states |

### Colores Neutrales
| Nombre | Hex | Uso |
|--------|-----|-----|
| Pure White | `#FFFFFF` | Surface base dominante |
| Pure Black | `#050505` | Texto principal, overlays |
| Warm Gray | `#F9FAFB` | Surface secundario |
| Ivory | `#F8F4EC` | Hover states alternativos |
| Beige | `#DFD2BF` | Paneles secundarios |

### Colores de Acento
| Nombre | Hex | Uso |
|--------|-----|-----|
| Safety Blue | `#0066FF` | Links, info badges |
| Voltage Yellow | `#FAFF00` | Warnings, estetica tech |
| Digital Red | `#FF2A2A` | Errores |
| Azul Pastel | `#A7D8F4` | CTAs secundarios |

### Colores de Feedback
| Estado | Hex |
|--------|-----|
| Success/Olive | `#9DB38B` |
| Error/Rust | `#B25E5E` |
| Warning | `#C4A882` |

## Tipografia

| Tipo | Fuente | Uso |
|------|--------|-----|
| Sans Default | Inter (variable) | Texto general |
| Display/Premium | Satoshi | Titulos, headings |
| Monospace | JetBrains Mono | Precios, VINs, datos tecnicos |

## Filosofia de Diseno
- **Tech Brutalism** + **Neon Accents**
- Alto contraste para accesibilidad (WCAG AA)
- Sistema de tokens centralizado
- Mobile-first con escala 8pt

## NO Usar
- Color azul como primario
- Referencias a Uber, DiDi, Cabify
- Terminologia de "conductores de apps"
- Emojis en materiales formales (usar SVGs)
