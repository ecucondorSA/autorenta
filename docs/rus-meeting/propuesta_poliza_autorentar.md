## 1. Modelo de Negocio: Comodato de Comunidad

AutoRentar **no es un alquiler tradicional**. Es un modelo de **comodato** (préstamo gratuito) donde:

- Los propietarios **prestan** sus vehículos a la comunidad
- Los usuarios pagan una **contribución** por el uso
- Los propietarios reciben **rewards mensuales** basados en participación, no por transacción

---

## 2. Distribución de Ingresos: Modelo 15/70/15

| Componente | % | Función |
|------------|---|---------|
| **AutoRentar** | 15% | Infraestructura, soporte, tecnología |
| **Pool de Propietarios** | 70% | Rewards mensuales distribuidos por puntos |
| **FGO** | 15% | Fondo de Garantía Operativa |

```
Contribución del Usuario = 15% AutoRentar + 70% Pool + 15% FGO
```

> [!IMPORTANT]
> **El propietario NO recibe pago directo por cada transacción.** Recibe rewards mensuales del Pool según su participación en la comunidad.

---

## 3. Sistema de Triple Capa de Protección

```
CAPA 1: Garantía del Usuario ($1,000 USD bloqueados)
    │   Cubre daños menores y franquicias
    │
    ▼
CAPA 2: FGO (15% de cada transacción)
    │   Absorbe siniestros operativos
    │   80% de incidentes se resuelven aquí
    │
    ▼
CAPA 3: Póliza RUS
        Solo siniestros de alta cuantía
```

**Beneficio para RUS:** La mayoría de incidentes no llegan a la aseguradora.

## 5. Prima del Seguro

### De dónde sale la prima:

| Fuente | Detalle |
|--------|---------|
| **Del 15% de AutoRentar** | La prima se descuenta del fee de plataforma |
| **NO es adicional** | El propietario no paga extra |
| **NO afecta el Pool** | El 70% de propietarios queda intacto |

### Tasas propuestas:

| Categoría | Prima (del booking) | Del 15% de AutoRentar |
|-----------|---------------------|----------------------|
| Económico (<$20K) | 2-3% | ~15-20% del fee |
| Estándar ($20K-$40K) | 3-4% | ~20-27% del fee |
| Premium (>$40K) | 4-5% | ~27-33% del fee |

### Ejemplo concreto:

| Concepto | Monto |
|----------|-------|
| Contribución del usuario | $100 USD |
| → AutoRentar (15%) | $15 USD |
| → Pool Propietarios (70%) | $70 USD |
| → FGO (15%) | $15 USD |
| **Prima RUS (3% del total)** | **$3 USD** (sale del $15 de AutoRentar) |
| **AutoRentar neto** | **$12 USD** |
