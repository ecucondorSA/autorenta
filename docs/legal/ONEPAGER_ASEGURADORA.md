# AUTORENTA - Informacion para Aseguradoras

**Documento Informativo - Version 1.0**
**Fecha:** Enero 2026

---

## ¿Que es Autorenta?

Autorenta es una **comunidad de comparticion de vehiculos** entre particulares, similar al modelo de BlaBlaCar para compartir viajes.

**NO somos:**
- Una empresa de alquiler de autos
- Una rent-a-car tradicional
- Una flota comercial

**SI somos:**
- Una comunidad de propietarios que prestan sus vehiculos ocasionalmente
- Una plataforma tecnologica que facilita el contacto entre particulares
- Un sistema de verificacion de documentos y seguros

---

## ¿Como funciona?

```
PROPIETARIO                         USUARIO
(Comodante)                        (Comodatario)
     |                                  |
     |    Contrato de COMODATO          |
     |    (Prestamo gratuito)           |
     +<-------------------------------->+
     |                                  |
     |    NO hay pago directo           |
     |    NO hay alquiler               |
     |    NO hay renta                  |
     +----------------------------------+
```

### Flujo simplificado:

1. **Propietario** ofrece su vehiculo en la comunidad
2. **Usuario** solicita el prestamo del vehiculo
3. Se firma **contrato de comodato** (prestamo gratuito)
4. Usuario usa el vehiculo para fines particulares
5. Usuario devuelve el vehiculo
6. **Propietario NO recibe pago** por el prestamo

---

## ¿Que recibe el propietario?

El propietario **NO RECIBE PAGO** por cada prestamo individual.

Puede recibir **rewards comunitarios** que:

| Caracteristica | Detalle |
|----------------|---------|
| **No son pago por uso** | No hay correlacion reserva → dinero |
| **Son diferidos** | Se calculan mensualmente |
| **Son variables** | Dependen de multiples factores |
| **Tienen tope** | Nunca superan los gastos del vehiculo |
| **No son exigibles** | El propietario no puede reclamarlos |

### Calculo de rewards (ejemplo):

| Concepto | Puntos | Vinculado al uso? |
|----------|--------|-------------------|
| Disponibilidad | 10/dia | NO |
| Antiguedad | 50/mes | NO |
| Buen rating | 100/mes | Parcial |
| Referidos | 200/ref | NO |

**Resultado:** El propietario puede recibir rewards **sin que su auto sea usado**.

## ¿Por que NO es alquiler?

| Criterio Legal | Alquiler (Art. 1187 CCyC) | Comodato Autorenta (Art. 1533 CCyC) |
|----------------|---------------------------|-------------------------------------|
| Contraprestacion | Precio cierto | **Ninguna** |
| Pago por uso | Si, por reserva | **No, $0 por reserva** |
| Correlacion | 1:1 directo | **No existe** |
| Lucro | Si | **No (rewards ≤ gastos)** |
| Naturaleza | Comercial | **Particular/Civil** |

## Requisitos de seguro

Para participar en la comunidad, el propietario **DEBE** tener:

| Requisito | Obligatorio | Verificacion |
|-----------|-------------|--------------|
| **Poliza vigente** | SI | Upload + validacion |
| **Clausula Conductor Indeterminado** | SI | Verificacion manual |
| **RC minimo $50M ARS** | SI | Verificacion en poliza |
| **Vigencia minima 60 dias** | SI | Validacion automatica |

### ¿Por que Conductor Indeterminado?

La clausula de **Conductor Indeterminado** permite que terceros con licencia vigente conduzcan el vehiculo. Esta clausula:

- Ya contempla el riesgo de terceros
- Ya esta tarifada en la prima (~5-15% adicional)
- Es la cobertura apropiada para prestamos a familiares, amigos, conocidos
- **No requiere poliza comercial ni de flota**

---

## Limites operativos

Para mantener el caracter **ocasional y no comercial**, la plataforma impone:

| Limite | Valor | Enforcement |
|--------|-------|-------------|
| Dias por mes | Max 15 | Sistema bloquea al exceder |
| Dias consecutivos | Max 5 | Sistema impide reservas largas |
| Rewards anuales | ≤ Gastos vehiculo | Calculo automatico |

Estos limites garantizan que:
- El uso es **ocasional**, no profesional
- El propietario **no lucra** con el vehiculo
- El caracter es **particular**, no comercial

## Documentacion disponible

Para cada prestamo, la plataforma genera y conserva:

1. **Contrato de comodato** firmado digitalmente
2. **Inspeccion pre-entrega** con fotos 360°
3. **Inspeccion post-devolucion** con fotos 360°
4. **Verificacion de licencia** del usuario
5. **Historial de uso** del vehiculo

Esta documentacion esta disponible para:
- El propietario
- La aseguradora (ante requerimiento formal)
- Autoridades judiciales (ante orden judicial)

## Preguntas frecuentes de aseguradoras

### "¿Es alquiler encubierto?"

**NO.** El propietario no recibe pago por el prestamo. Los rewards comunitarios:
- No estan vinculados a reservas individuales
- No superan los gastos del vehiculo
- No constituyen renta ni precio

### "¿Agrava el riesgo?"

**NO**, si tiene Conductor Indeterminado. Esta clausula ya contempla y tarifa el riesgo de que terceros conduzcan. El uso ocasional (max 15 dias/mes) esta dentro de los parametros normales de la clausula.

### "¿Debo informar a mi aseguradora?"

**RECOMENDAMOS** que el propietario informe a su productor de seguros. La plataforma provee un template de comunicacion y permite subir evidencia de la notificacion.

### "¿Que pasa en un siniestro?"

El procedimiento es el normal:
1. Propietario denuncia a su aseguradora
2. Aseguradora investiga (puede solicitar documentacion a la plataforma)
3. Cobertura aplica segun clausula Conductor Indeterminado
4. Plataforma colabora con documentacion si es requerida

## Modelo economico

```
Usuario paga: $100 USD (ejemplo)
         |
         +---> 75% Pool comunitario (rewards diferidos)
         |
         +---> 15% Fee operativo (Autorenta)
         |
         +---> 10% Fondo de Garantia Operativa

Propietario recibe por esta reserva: $0 USD
```

**Nota:** El 75% va a un pool mensual que se distribuye por puntos, NO por uso individual.
