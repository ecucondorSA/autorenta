# Test Booking Flow

Ejecuta un test completo del flujo de booking en el navegador.

## Instrucciones

1. Abre Chrome y navega a la app (localhost:4200 o produccion)
2. Ve al marketplace y selecciona un auto
3. Completa el flujo de reserva hasta el checkout
4. Toma screenshots en cada paso
5. Verifica la consola por errores
6. Reporta cualquier problema de UX encontrado
