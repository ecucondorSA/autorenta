# Mobile Audit

Ejecuta una auditoría completa de la versión móvil.

## Instrucciones

1. Redimensiona el navegador a 375x812 (iPhone X)
2. Navega por las páginas principales:
   - Home/Marketplace
   - Car Detail
   - Booking Flow
   - Profile
   - Wallet
3. Verifica:
   - Touch targets (min 44x44px)
   - Scroll horizontal (no debe existir)
   - Legibilidad de texto
   - Funcionamiento de bottom sheets
4. Toma screenshots de cada página
5. Reporta issues encontrados

## Output
- Screenshots responsive
- Lista de issues de accesibilidad
- Recomendaciones de mejora
