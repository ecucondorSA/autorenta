# Autorenta
### El Airbnb de los autos para el Cono Sur

---

## El Problema

| Situación | Impacto |
|-----------|---------|
| 40% de los autos particulares se usan <2 horas/día | Capital dormido |
| Rent-a-cars tradicionales: caros, burocracia, filas | Mala experiencia para turistas |
| Dueños de autos premium no tienen cómo monetizar | Oportunidad perdida |

---

## La Solución: Autorenta

Plataforma P2P que conecta **dueños de vehículos** con **personas que necesitan alquilar**.

```
DUEÑO                          CLIENTE
  │                               │
  │  Publica su auto              │  Busca auto
  │  Define precio y fechas       │  Reserva y paga
  │  Recibe 85% del alquiler      │  Retira el auto
  │                               │
  └───────────[ AUTORENTA ]───────┘
              15% comisión
              Seguro incluido
              Soporte 24/7
```

## Ventaja Competitiva

### vs Rentennials (principal competidor)

| Aspecto | Rentennials | Autorenta |
|---------|-------------|-----------|
| Comisión | 20% | **15%** |
| Seguro | Dueño paga | **Incluido** |
| Seguridad de datos | Vulnerable* | Diseño seguro |
| Foco | Autos económicos | **Premium + económicos** |

*Nota: Análisis técnico identificó exposición de datos personales de 254 usuarios.
